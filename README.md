## Run locally

1. Clone this repo (or download and extract).
2. Open `index.html` in a browser, or use a simple local server, e.g.:
   ```bash
   # Python
   python -m http.server 8000
   # or Node (npx)
   npx serve .
   ```
3. Visit `http://localhost:8000` (or the port shown).

## Add this portfolio to your GitHub & deploy with GitHub Pages

1. **Create a new repository** on GitHub (e.g. `vcard-personal-portfolio` or `portfolio`).
2. **Point this project to your repo** (replace `YOUR_USERNAME` and `YOUR_REPO` with your GitHub username and repo name):
   ```bash
   git remote set-url origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
   # Or if you prefer to add a new remote:
   # git remote add mygithub https://github.com/YOUR_USERNAME/YOUR_REPO.git
   ```
3. **Push your code:**
   ```bash
   git add .
   git commit -m "My portfolio with custom content"
   git push -u origin master
   ```
4. **Turn on GitHub Pages:**
   - In your repo on GitHub: **Settings** → **Pages**.
   - Under **Source**, choose **Deploy from a branch**.
   - Branch: **master** (or **main**), folder: **/ (root)**.
   - Save. Your site will be at `https://YOUR_USERNAME.github.io/YOUR_REPO/`.
